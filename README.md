# Concert Ticket Hub

Android app for SDC 340 - 3.4 Performance Assessment (Concert Ticket Mobile App)

**Author:** Patrick Gonzalez

## Description

Concert Ticket Hub lets a user pick a concert event from a dropdown, enter the number of tickets they want, and calculates the total cost.

## Features

- Custom app icon (ticket + music note design)
- Spinner with 3 events: Pop Music Festival ($49.99), Jazz Music Festival ($45.99), Country Music Festival ($42.99)
- Numeric input for ticket count with validation
- Button to calculate and display the total cost

## Tech

- Kotlin
- ConstraintLayout
- Android SDK (minSdk 24, targetSdk/compileSdk 36)
